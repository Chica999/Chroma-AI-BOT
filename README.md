## ✦ What is Chroma AI?

</div>

> **Chroma AI** is a private Discord bot built to combine AI utilities, moderation, automation, verification, and advanced server protection into one unified system. Forget juggling five bots — Chroma handles it all with smart tools, cleaner server management, and hardened security.
